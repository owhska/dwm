| Tecla / Mod               | Função                | Comando / Observação                          |
| ------------------------- | --------------------- | --------------------------------------------- |
| `MOD` + `Enter`           | Abrir terminal        | `st`                                          |
| `MOD` + `p`               | Abrir dmenu           | `dmenu_run`                                   |
| `MOD` + `Shift` + `c`     | Fechar janela         | Fecha a janela ativa                          |
| `MOD` + `j` / `k`         | Mover foco            | Próxima/anterior janela                       |
| `MOD` + `h` / `l`         | Ajustar master        | Diminuir/aumentar tamanho master              |
| `MOD` + `Shift` + `Enter` | Trocar master         | Coloca janela selecionada como master         |
| `MOD` + `t` / `f` / `m`   | Trocar layout         | Tile / Floating / Monocle                     |
| `MOD` + `1–9`             | Mudar tag             | Alterna para tag correspondente               |
| `MOD` + `Shift` + `1–9`   | Mover janela para tag | Move a janela ativa para tag correspondente   |
| `XF86AudioRaiseVolume`    | Aumentar volume       | `pamixer --allow-boost -i 5`                  |
| `XF86AudioLowerVolume`    | Diminuir volume       | `pamixer --allow-boost -d 5`                  |
| `XF86AudioMute`           | Mute                  | `pamixer -t`                                  |
| `XF86MonBrightnessUp`     | Aumentar brilho       | `brightnessctl set +10%`                      |
| `XF86MonBrightnessDown`   | Diminuir brilho       | `brightnessctl set 10%-`                      |
| `MOD` + `b`               | Abrir browser         | `firefox` (adicionado no config.h)            |
| `MOD` + `Shift` + `r`     | Restart dwm           | Recarrega dwm após recompilação (patch comum) |
